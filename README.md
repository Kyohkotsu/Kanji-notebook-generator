# Introduction 
Le projet vise à développer un programme qui génère automatiquement un fichier pdf servant à l'apprentissage de kanji (caractères chinois employés dans la langue japonaise).

# Getting Started
Télécharger le code source en .py et télécharger au besoin les librairies manquantes à votre environnement.
Exécuter main.py 
Saisir les kanjis dans la boîte de texte.
Clique sur le bouton de recherche.
